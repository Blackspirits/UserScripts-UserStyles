## UserScripts

### [IMDb Ad/Pro Cleaner]

#### [1.4.1] - 2024-01-04
##### Current Version
- Blocks ads/sponsored content and strips IMDbPro UI
- Safe zone protection for credit accordions
- Performance optimizations with requestAnimationFrame throttling
- Multi-language support (English, Portuguese sponsored content)

### [Torrenting.com Auto Posters]

#### [2.0.0] - 2024-01-04
##### Current Version
- Shows posters in torrent listings
- Supports multiple page types (featured, browse, requests, torrent details)
- Fallback empty-image SVG for missing posters
- Lazy loading and async decoding for performance
- Same-origin fetch for privacy
